## [0.1.0] - 2026-02-10

### 추가

#### 핵심 기능
- **실시간 포트 스캔**: 시스템 포트 사용 현황을 2초 이내에 표시
- **TUI 인터페이스**: Bubbletea 기반 직관적인 터미널 UI
- **Docker 통합**: 컨테이너 포트 자동 감지 및 시각적 표시
- **스마트 추천 시스템**: 자주 종료하는 프로세스 자동 표시
- **검색 및 필터링**: 실시간 검색으로 빠른 포트 찾기
- **안전한 프로세스 종료**: SIGTERM → SIGKILL 단계적 종료
- **히스토리 추적**: SQLite 기반 종료 이력 관리

#### TUI 기능
- Vim 스타일 키보드 네비게이션 (`j`, `k`, `gg`, `G`)
- 키보드 단축키 지원 (`/` 검색, `d` Docker 필터, `h` 히스토리)
- 실시간 검색 및 필터링
- 프로세스 종료 확인 다이얼로그
- 상태바 및 도움말 화면

#### 프로세스 관리
- 단계적 프로세스 종료 (SIGTERM → 3초 대기 → SIGKILL)
- 시스템 중요 프로세스 보호 (PID < 100)
- Docker 컨테이너 프로세스 종료 지원
- 프로세스 실행 상태 확인

#### 데이터 모델
- 포트 정보 모델 (PortInfo)
- 히스토리 엔트리 모델 (HistoryEntry)
- Docker 정보 모델 (DockerInfo)
- 일반 포트 및 시스템 포트 식별

#### 저장소
- SQLite 기반 히스토리 저장소
- WAL 모드로 동시 읽기/쓰기 지원
- 자동 마이그레이션
- 실패 시 로깅만 수행 (TUI 동작 유지)

#### UI 컴포넌트
- 헤더 컴포넌트 (앱 이름, 포트 수)
- 포트 목록 컴포넌트 (스크롤, 하이라이트)
- 상태바 컴포넌트 (단축키 가이드)
- 검색 바 컴포넌트
- 다이얼로그 컴포넌트

#### 플랫폼 지원
- macOS 프로세스 관리
- Linux 프로세스 관리
- Windows 프로세스 관리

#### 감지 기능

- Docker SDK + CLI fallback
- 컨테이너 정보 확장 (ContainerID, ContainerName, ImageName)

### 제공된 키보드 단축키

#### 네비게이션
- `↑` / `k`: 위로 이동
- `↓` / `j`: 아래로 이동
- `gg`: 맨 위로 이동
- `G`: 맨 아래로 이동

#### 작업
- `Enter`: 프로세스 종료
- `/`: 검색 모드
- `d`: Docker 필터 토글
- `h`: 히스토리 보기
- `?`: 도움말
- `r`: 새로고침
- `q` / `Ctrl+C`: 종료

### 마커 시스템
- `[D]`: Docker 컨테이너에서 실행 중
- `[!]`: 최근 30일간 3회 이상 종료됨
- `⚠`: 시스템 중요 프로세스

### 성능
- 2초 이내 포트 스캔
- 100ms 이내 검색 필터링
- 효율적인 UI 렌더링

### 보안
- 시스템 중요 프로세스 보호
- 안전한 프로세스 종료
- 사용자 확인 다이얼로그

### 호환성
- macOS, Linux, Windows 지원
- Docker Desktop 및 Docker Engine 지원
- 터미널 호환성 (ANSI 지원)
